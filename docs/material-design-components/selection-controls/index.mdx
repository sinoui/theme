# 选择控件 Selection controls

选择控件允许用户选择选项。

## 使用 Usage

选择控件使用户可以完成涉及选择的任务，例如选择选项或者打开、关闭设置。选择控件位于要求用户做出决定或声明诸如设置或对话框等首选项的屏幕上。

### 何时使用单选按钮 When to use radio buttons

单选按钮可用于：

- 从列表中选择一个选项
- 公开所有可用选项

如果可以折叠可用选项，请考虑使用下拉菜单代替，因为它占用更小的空间。

</ImageWithTitle>

### 父子关系复选框 Parent and child checkboxes

复选框之间可以是父子关系：

- 当父复选框被选中，则所有子复选框也被选中
- 当父复选框取消选中时，则所有子复选框也被取消选中
- 如果只是部分而不是全部子复选框被选中，那么父复选框将变为不确定复选框

</ImageWithTitle>

### Behavior 行为

当用户切换开关时，其相应的动作将立即生效。如果无法打开开关，则开关将自动关闭。

import switchesToggle from './images/slidercontrols-1a-v02.mp4';

<ImageWithTitle video={switchesToggle}>无法打开的开关</ImageWithTitle>

#### 显示处理进度 Display processing status

因为开关显示事物的实际状态，有时候状态会有所延迟。在这种情况下，可以显示处理进度动画。

处理进度是在开关指示器上的动画。例如，当控制硬件功能的开关在确认其最终状态之前经历延迟时，可以使用它。

import switchesProcessing from './images/slidercontrols-1b-v02.mp4';

<ImageWithTitle video={switchesProcessing}>处理进度动画</ImageWithTitle>

### 状态 States

开关可以打开或关闭。开关有启用、悬停、聚焦和按下状态。

仅在使用触摸的形状上显示外部径向反应。

对于台式机，不需要径向反应。
