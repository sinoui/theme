# 文本字段 Text fields

文本字段允许用户输入、编辑文本。

## 使用 Usage

用户可以借助文本字段在界面上输入、编辑文本。它们一般会出现在表单和对话框中。

### 类型 Types

有两种类型文本字段：

- 填充模式文本字段
- 轮廓模式文本字段

两种类型的文本字段都使用容器为交互提供清晰的边界，从而使输入框在布局中可发现。

### 选择正确的文本字段 Choosing the right text field

两种类型的文本字段提供一样的功能。所以仅从样式上做出选择即可。

选择哪一种从以下角度考虑：

- 与您的应用风格匹配
- 适合您的 UI 目标
- 与其他组件（如按钮）和周围内容形成对比

### 标签文本 Label text

标签文本是用来给用户指示文本字段需要输入哪些信息的。每个字段应有一个标签。

标签文本应与输入行对齐，并始终可见。可以将其放在文本字段的中间，也可以放在容器顶部附近。

- 如果大部分字段是必填的，请在可选字段的标签文本旁边的括号中显示“可选”来指示其可选

- 如果有多个错误，则错误文本描述如何避免最可能的错误

## 输入类型 Input types

文本字段有以下三种输入方式：

- **单行文本字段**仅显示一行文本
- **多行文本字段**增长以容纳多行文本
- **文本域**是一个固定高度的字段

  </p>

## 研究 Research

Material Design 发起了约 600 人参与的两项研究，来评估第一版文本字段设计。这项研究形成了当前这个版本的文本字段设计，包括以下最重要的可用性上的改进：

- 带有矩形的封闭文字字段要优于带有线条的文本字段
- 文本字段应该有半透明度填充和底部线条，或者全透明的填充和不透明的边框
- 文本字段的线条或边框颜色与背景色应形成最小 3:1 的对比度
- 标签文本应显示在文本字段框内部
- 文本字段应有圆角
