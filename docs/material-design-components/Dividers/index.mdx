# 分隔线 Dividers

分隔线是一个在列表和布局中分组内容的细线。

![divider-intro](images/divider-intro.png)

## 使用 Usage

分隔线将内容分成清晰的组。

## 类型 Types

### 全宽分隔线 Full-bleed dividers

全宽分隔线将内容分成不同的部分，例如：

- 分割列表和布局元素
- 指示元素可能在哪里扩展

import fullBleed from './images/dividers-full-bleed.png';
import baselineDrawer from './images/baseline-drawer.png';

<Row gutter={48}>
  <Column xs={24} sm={12}>
    <ImageWithTitle img={fullBleed}>
      如果列表没有锚定元素（例如图标），则留白可能不足以分割内容。全宽分隔线可以帮助分离出单个内容。
    </ImageWithTitle>
  </Column>
  <Column xs={24} sm={12}>
    <ImageWithTitle img={baselineDrawer}>
      导航抽屉中的全宽分隔线。
    </ImageWithTitle>
  </Column>
</Row>

### 插入分隔线 Inset dividers

插入分隔线将相关内容分开，例如在邮箱中的邮件。它们应与锚定元素（例如图标或者头像）一起使用，并与应用栏标题左对齐。
