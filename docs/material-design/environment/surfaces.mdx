# 表面 Surfaces

Material Design 具有三维质感，这体现在其表面（Surfaces）、深度（depth）和阴影（shadows）上。

## 材料环境 Material Environment

### 物理世界 The physical world

在物理世界中，物品可以堆叠或附着在一起，但不能互相穿越。它们投下阴影并反射光线。

Material Design 应用了这些特征，设计了表面如何在 UI 中显示和移动。表面在三维世界中的移动方式类似如它们在物理世界中的移动方式。可以将该空间模型一致地应用到应用程序中。

### 深度 Depth

UI 展示在由光、表面和投射阴影构建的三维空间中。UI 中的所有元素都可以水平移动、垂直移动或者沿着 Z 轴以不同深度进行移动。深度表示元素在正 Z 轴朝向观察者的位置。

## 属性 Attributes

### 基准材料表面 Basic Material surface

基准材料表面是不透明白色的，1dp厚度，并且投射阴影。Material Design 中所有的 UI 元素均来自对这个基准表面的修改。

### 行为 Behavior

材料表面有以下行为方式：

* **刚性表面**在所有相互作用中均保持相同大小。
* **可拉伸表面**可以沿着一个或多个边缘伸展或者收缩，直至达到尺寸限制，然后表现为刚性表面。
* **可平移的表面**在交互过程中保持相同大小。可以滚动显示超出表面的内容，直至达到内容边界（即可平移表面可以有滚动条）。当滚动到内容边界，表面在该滚动方向上就是刚性的。（备注：“可平移的表面”是直译 Pannable surfaces 而来的，表示的是可以滚动其内部内容的表面。pan 是移动端手势术语，相当于拖拽，翻译过来就是平移。在移动端，需要用手指按住并移动界面，才能看到界面之外的内容。）

### 可拉伸表面 Stretchable surfaces

可拉伸表面可以在水平、垂直或者两个方向上伸展，直至达到其尺寸限制。达到尺寸限制后，表面将变得刚性。

通常，用户与表面的交互会引起单方向上的伸展。例如，点击“更多细节”图标可引起卡片在垂直方向上展开并显示额外的内容。

### 表面不透明度 Surface opacity

材料表面可以是透明的、半透明的或者不透明的。

可能需要处理透明或半透明表面的背景，以保持在表面上的文字的可读性。

透明表面缺少清晰的边缘，因此很难确定表面的开始和结束位置，以及内容属于哪个表面。

### 遮罩层 Scrim

遮罩层是应用于材料表面的临时处理，目的是使表面上的内容不那么突出。它们有助于引导用户注意屏幕的其他部分，忽略界面上处理过的信息。

可以有多种方式使用遮罩层，包括：

* 使表面和其内容变暗或者变淡
* 降低表面和其内容的不透明度

在屏幕上的多个表面可以同时显示遮罩层。遮罩层可以显示在任何海拔（深度）上，无论是在前景还是背景中。
