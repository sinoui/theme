# 状态 States

状态是传达组件或者交互元素的视觉表示方式。

## 使用 Usage

用户界面元素与用户之间的交流是通过状态来表示的。每种组件的状态需要在视觉上相似但组件本身没有过大的改变，与此同时，当前状态要与其他状态及周围的布局有明显的区别。

### 叠加 Overlay

在一个元素上方叠加半透明图层来代表状态，它提供了一种使用不透明度可视化状态的系统方法。

叠加可以被应用于整个元素或在元素上方叠加半透明的圆形。叠加的颜色需要与元素本身的文字及图表颜色相符，如果再改变状态后文字和图标的颜色产生了变化，那叠加的颜色应该与文字和图标最后显示的颜色相符。

## 禁用状态 Disabled

### 使用 Usage

禁用状态代表组件或元素是不可交互的，且在 UI 设计中不需要着重强调。禁用状态以启用状态的 38% 不透明度显示。

颜色的改变和降低海拔也可以用来表示禁用状态。
