# 状态 States

状态是传达组件或者交互元素的视觉表示方式。

## 使用 Usage

用户界面元素与用户之间的交流是通过状态来表示的。每种组件的状态需要在视觉上相似但组件本身没有过大的改变，与此同时，当前状态要与其他状态及周围的布局有明显的区别。

### 叠加 Overlay

在一个元素上方叠加半透明图层来代表状态，它提供了一种使用不透明度表达可视化状态的系统方法。

叠加可以被应用于整个元素或在元素上方叠加半透明的圆形。叠加的颜色需要与元素本身的文字及图标的颜色相符，如果在改变状态后文字和图标的颜色产生了变化，那么叠加的颜色应该与文字和图标变化后的颜色相符。

## 禁用状态 Disabled

### 使用 Usage

禁用状态代表组件或元素是不可交互的，且在 UI 设计中不需要着重强调。禁用状态以启用状态的 38% 不透明度显示。

颜色的改变和降低海拔也可以用来表示禁用状态。

有些组件可以继承禁用状态，例如切换按钮、选择控件、文本输入框、图标、网格和列表项、卡片、纸片和按钮。

## 激活状态 Activated

叠加图层、颜色改变或者其他应用在组件某个部分或元素上的视觉方案，均可用来表示激活状态。

## 按下 Pressed

### 使用 Usage

按下状态是指用户使用手指、鼠标、键盘或声音等输入方式触发轻触或点击操作。按下状态可应用于所有可交互组件上。

按下状态引起了一组内容的改变，并且强调程度为高等级。

波纹叠加可表示按下状态。它可以应用于整个组件或组件中的元素，也可以应用于组件的一部分上的圆形。
